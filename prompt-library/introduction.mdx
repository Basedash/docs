The Basedash prompt library provides ready-to-use prompts for Chat, Charts, and Reports. Copy these prompts and customize them for your specific data and use cases.

## How to use this library

Each prompt is designed to work with Basedash AI features. Simply copy the prompt and paste it into the appropriate feature:

- **Chat prompts**: Use in the [Chat interface](/features/chat) for ad-hoc data analysis
- **Chart prompts**: Use when creating [Charts](/features/charts) for visualizations
- **Report prompts**: Use when setting up [scheduled Reports](/features/reports)

## Customizing prompts

All prompts can be customized to match your data:

- Replace table and column names with your actual schema
- Adjust time periods to fit your needs
- Modify filters and conditions for your use case
- Add or remove specific metrics
