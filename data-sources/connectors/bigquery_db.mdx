## Overview

BigQuery is available as a data source via Basedash Warehouse. This allows you to sync your BigQuery data into a centralized warehouse managed by Basedash.

## Setup

1. In Basedash, open the command menu (âŒ˜K) and open "Data sources"
2. Select "Add data source"
3. Search for "BigQuery"
4. Select BigQuery and follow the setup flow through Fivetran

## Table and column descriptions

Basedash automatically pulls in table and column descriptions from your BigQuery database. These descriptions provide valuable context to the AI about your data structure and are used to generate more accurate queries and insights.

## Documentation

For detailed information about the BigQuery connector, including schema information and sync behavior, see the [Fivetran BigQuery documentation](https://fivetran.com/docs/connectors/databases/bigquery).
