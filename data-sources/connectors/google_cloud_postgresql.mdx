## Overview

Google Cloud SQL for PostgreSQL is available as a data source via Basedash Warehouse. This allows you to sync your Google Cloud SQL for PostgreSQL data into a centralized warehouse managed by Basedash.

3. Search for "Google Cloud SQL for PostgreSQL"
4. Select Google Cloud SQL for PostgreSQL and follow the setup flow through Fivetran
