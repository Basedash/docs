## Overview

Azure Cosmos DB for MongoDB is available as a data source via Basedash Warehouse. This allows you to sync your Azure Cosmos DB for MongoDB data into a centralized warehouse managed by Basedash.

2. Select "Add data source"
3. Search for "Azure Cosmos DB for MongoDB"
4. Select Azure Cosmos DB for MongoDB and follow the setup flow through Fivetran
